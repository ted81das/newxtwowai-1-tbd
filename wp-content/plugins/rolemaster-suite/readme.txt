== Description ==

[RoleMaster Suite](https://jeweltheme.com/user-role-editor) a User Role Editor plugin for WordPress that allows website administrators with the ability to fine-tune user roles and capabilities with precision.

Designed for flexibility and easy of use, it is the perfect tool for managing access control on your WordPress site. With its user-friendly interface and rich feature set, this plugin simplifies the process of user role management, saving you valuable time and effort.

== 🚀 Why RoleMaster Suite? ==

1. **Custom User Role Creation & Deletion**: Easily create new user roles and remove them anytime when it’s not needed.

2. **Capability Management**: Add new capabilities to roles or delete them to maintain a secure and organized site structure.

3. **Duplicate User Roles**: Easily clone existing roles, saving time and effort when setting up similar capabilities.

4. **Role Renaming**: Update user role names to reflect their purpose or organizational changes.

5. **Intuitive User Interface (UI)**: RoleMaster Suite's user-friendly interface ensures a smooth experience for administrators.

6. **User Level Selection**: Assign users to different levels, ensuring users have access to the features they need.

7. **Ecommerce & Membership Ready**: Ideal for ecommerce platforms and membership sites requiring specialized access levels.

8. **Enhanced Role Customization**: Modify existing roles by adding new capabilities to meet your site's unique needs.

9. **One-Click Capability select & deny**: Quickly assign or revoke capabilities with a single click, for faster role management.

== Installation ==
1. Upload the 'rolemaster-suite' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'Settings > RoleMaster Suite' to configure User Role Editor options.

== Frequently Asked Questions ==
= Can I revert to the default settings? =
Yes, you can easily revert to the default settings by deactivating the plugin.

= Is it compatible with all WordPress themes & plugins? =
RoleMaster Suite is designed to be compatible with most WordPress themes and plugins. However, in rare cases, some themes and plugins may have conflicting styles or customizations that could affect the appearance of the plugin, but the functionality will works properly.
